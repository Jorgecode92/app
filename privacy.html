Privacy Policy

Space Invaders: Defense Grid

This game stores local gameplay progress such as best score, coins, unlocks, settings, and reward claim status. In browser mode this is stored in localStorage. In TikTok mode this uses the TikTok Mini Games storage API.

TikTok silent login may provide a temporary authorization code. The base version does not send that code to a backend. If backend sync is added later, this policy must be updated before release.

Rewarded ads are shown only when the player chooses revive or bonus reward actions. TikTok may process ad and platform data according to its own policies.
