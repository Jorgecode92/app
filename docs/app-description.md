# App Description Draft

Space Invaders: Defense Grid is a quick-play arcade defense shooter built for short mobile sessions. Drag to steer, fire through falling invader waves, protect shield lanes, earn coins, and unlock cosmetic ship, laser, and shield upgrades.

Key features:

- Portrait-first arcade play
- 30 to 90 second run loop
- Score, lives, waves, and shield pressure
- Coins from kills, wave clears, daily rewards, and run summaries
- Optional rewarded revive once per run
- Optional post-run rewarded coin bonus
- Unlockable cosmetics and small arcade-balanced upgrades

Short store copy:

Hold the defense line in fast arcade waves. Earn coins, unlock new looks, revive once, and chase your best score in a TikTok-ready mini game built for quick replays.
